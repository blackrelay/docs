This page is a practical link index. It is not a source of truth by itself.

Prefer official documentation and contract source for transaction behaviour. Treat unofficial pages and tools as working notes that still need verification.

## Official EVE Frontier

Link | Use
--- | ---
[Builder Documentation](https://docs.evefrontier.com/) | Official concept, builder and tool documentation.
[World Resources](https://docs.evefrontier.com/tools/resources) | Current World API and package resource links.
[World Contracts](https://github.com/evefrontier/world-contracts) | Contract source, module names and event shapes.
[Builder Scaffold](https://github.com/evefrontier/builder-scaffold/tree/main) | Local examples, scripts and extension starting point.
[EVE Vault Releases](https://github.com/evefrontier/evevault/releases/) | Wallet extension and app releases.
[dApp Kit](https://docs.evefrontier.com/tools/dapp-kit) | Wallet-facing dApp hooks and GraphQL helpers.

## Black Relay

Black Relay belongs after the official and direct sources in the stack:

Link | Use
--- | ---
[Public API](https://api.blackrelay.network/v1) | Source-aware public read layer.
[Registry Releases](https://github.com/blackrelay/registry/releases/latest) | Static-client and Registry import/export tooling.
[Registry Repository](https://github.com/blackrelay/registry) | Canonical data pipeline source.

Use Black Relay when you need a public read model not when you need to sign, mutate, deploy or prove final live object state.
